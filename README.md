# 👶 Kidz Commerce - Premium Educational Toys Store

Kidz Commerce is a modern, high-performance E-commerce platform built for high-quality educational toys and learning materials. It features a seamless user experience, secure checkout, and a robust order management system.

[![Site Live](https://img.shields.io/badge/Live-Demo-brightgreen?style=for-the-badge)](https://kidz-commerce.vercel.app)
[![Next.js](https://img.shields.io/badge/Next.js-14-black?style=for-the-badge&logo=next.js)](https://nextjs.org/)
[![TailwindCSS](https://img.shields.io/badge/Tailwind-CSS-blue?style=for-the-badge&logo=tailwind-css)](https://tailwindcss.com/)

---

## ✨ Key Features

- **🛍️ Smart Shopping Cart:** Interactive cart with dynamic quantity updates and real-time subtotal calculation.
- **🔐 Secure Authentication:** User authentication powered by **NextAuth.js** with Google & Email providers.
- **🚛 COD Order System:** Optimized Cash on Delivery (COD) checkout process with automated order ID generation.
- **📦 Order Tracking:** A dedicated "My Orders" dashboard for users to track their purchase history and status.
- **📱 Fully Responsive:** Beautifully crafted UI using **DaisyUI** and **Tailwind CSS**, optimized for Mobile, Tablet, and Desktop.
- **⚡ Server-Side Excellence:** Built with **Next.js 14 Server Actions** and MongoDB for blazing-fast data fetching.

---

## 🛠️ Tech Stack

- **Frontend:** [Next.js 14](https://nextjs.org/) (App Router), [React](https://reactjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/), [DaisyUI](https://daisyui.com/)
- **Backend:** [Next.js Server Actions](https://nextjs.org/docs/app/building-your-application/data-fetching/server-actions-and-mutations)
- **Database:** [MongoDB](https://www.mongodb.com/) (using Native Driver)
- **Authentication:** [NextAuth.js](https://next-auth.js.org/)
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/)
